# 🎮 Retro Tetris Game

A fully functional **Retro Style Tetris Game** built using **HTML, CSS, and JavaScript** with an arcade-inspired neon design and smooth gameplay mechanics.

---

## ✨ Features

- 🕹️ Classic Tetris gameplay
- 🌈 Retro neon arcade UI
- 📈 Score and level system
- ⚡ Increasing game speed with levels
- 🔄 Piece rotation system
- 💥 Line clearing mechanics
- 🎯 Collision detection
- ⌨️ Keyboard controls
- ☠️ Game over screen
- 🔁 Restart functionality
- 📱 Responsive centered layout

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Canvas API**

## 🎮 Controls

| Key | Action |
|------|---------|
| ⬅️ Left Arrow | Move Left |
| ➡️ Right Arrow | Move Right |
| ⬇️ Down Arrow | Soft Drop |
| ⬆️ Up Arrow | Rotate Piece |
| Spacebar | Hard Drop |

---

## 🧩 Game Mechanics

- Clear horizontal lines to gain points
- Each cleared line increases your score
- Speed increases as levels progress
- Game ends when blocks reach the top
